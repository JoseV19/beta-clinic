## 2. ESTRATEGIA DE PRECIOS

### Principios de Pricing

1. **Value-based, no cost-based**: El precio refleja CUANTO TIEMPO le ahorramos al medico, no cuanto nos cuesta el servidor
2. **Anchoring**: El plan medio debe parecer la mejor oferta
3. **Decoy effect**: El plan basico existe para hacer que el medio se vea superior

---

### Justificacion Psicologica del Pricing

**Plan Solo ($19/mes):**
- Funciona como *decoy*. Es barato pero limitado a 1 especialidad y 100 pacientes
- El medico que crece rapidamente sentira la friccion y upgradeara
- $19 es el precio de 2 cafes semanales — facil de justificar

**Plan Clinica ($49/mes) — EL GANADOR:**
- Desbloquea TODO: todas las especialidades, IA ilimitada, telemedicina, RIPS
- A $49/mes por clinica (no por doctor), es absurdamente barato vs la competencia
- Este es el plan que queremos que el 70% de usuarios elija
- Badge "Popular" + highlight visual en la landing

**Plan Hospital ($149/mes):**
- Existe para hacer que $49 parezca una ganga (*price anchoring*)
- Tambien captura el segmento enterprise con white-label y API
- El margen aqui es alto y los clientes son sticky (dificil de migrar)

### Estrategias de Conversion

1. **Trial Generoso**: 14 dias gratis del plan Clinica (no del Solo). Que prueben lo mejor

3. **Founder's Deal**: Primeras 50 clinicas obtienen plan Clinica a $29/mes DE POR VIDA
4. **Money-back**: 30 dias de garantia sin preguntas

## 3. COMPARATIVA CON COMPETIDORES

### El Panorama del Software Medico Tradicional

La mayoria del software medico en LATAM tiene estos problemas:

| Aspecto | Software Tradicional | Beta Clinic (Protocolo Omega) |
|---------|---------------------|-------------------------------|
| **Arquitectura** | Instalado en 1 PC (on-premise) | Cloud-native (Vercel + Supabase) |
| **Acceso** | Solo desde la computadora de la clinica | Cualquier dispositivo, cualquier lugar |
| **Velocidad** | 5-15 segundos por pantalla | < 400ms (React 19 + Vite 7) |
| **Diseno** | Windows XP aesthetic, botones grises | Protocolo Omega: oscuro, cinematico, premium |
| **Especialidades** | 1 sola, o modulos extra $$$$ | 4 nativas: General, Dental, Pediatria, Nutricion |
| **IA** | Inexistente | Gemini Flash: dietas, sugerencias, automatizacion |
| **Telemedicina** | Zoom aparte (otro pago) | Jitsi integrado, zero config |
| **Portal del Paciente** | No existe | App web con QR, citas online, recetas |
| **Actualizaciones** | Manuales, cobradas | Automaticas, incluidas |
| **Soporte** | Linea telefonica 9-5 | Chat + email, documentacion completa |
| **Precio** | $50-200/mes + instalacion + mantenimiento | Desde $19/mes, todo incluido |
| **Contrato** | Anual obligatorio | Mensual, cancela cuando quieras |
| **Backup** | "Esperemos que el tecnico hizo backup" | Automatico, encriptado, en la nube |
| **Mobile** | No | 100% responsive |

---

### Por que Nuestra Arquitectura es Superior

**1. Vercel (Frontend Hosting)**
- Edge network global: el site carga desde el servidor mas cercano al usuario
- Zero-downtime deploys: actualizamos sin que nadie lo note
- SSL automatico, DDoS protection incluida

**2. Supabase (Backend + DB)**
- PostgreSQL enterprise-grade
- Row Level Security: cada clinica solo ve SUS datos
- Realtime subscriptions: cambios se reflejan al instante
- Auth + Storage incluidos

**3. React 19 + Vite 7 (Frontend)**
- Code splitting automatico: solo carga lo que necesita
- HMR en desarrollo: iteramos 10x mas rapido que la competencia
- TypeScript end-to-end: menos bugs, mas estabilidad

**4. Clerk (Autenticacion)**
- Auth enterprise-grade sin construirla nosotros
- MFA, social login, magic links
- Session management, device tracking

**5. Google Gemini (IA)**
- Modelo Flash: rapido y gratuito
- Genera planes nutricionales en segundos
- Escalable a diagnosticos asistidos, transcripcion de consultas
